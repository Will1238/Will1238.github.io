I. Introduce

A. Multiple-output Guassian Processes

  • The standard GP model assumes a signle output variable only

  • The conventianal(传统的) approach for multi-output regression consists of applying an independent GP to each output variable

    • A suboptimal solution: the cross correlation among the output variables is not taken into account

  • One straightforward way for generalizing single-output GP regression to the multi-output case is to directly make the GP priors for different output variables correlated with one another（将单输出广义回归推广到多输出情况的一个简单方法是直接使不同输出变量的广义回归先验相互关联）

  • An alternative method relies on convolving the latent GPs with different kernels to produce multiple outputs and introduce cross correlation among them（另一种方法是将潜在高斯过程与不同的内核进行卷积，以产生多个输出，并在它们之间引入互相关）

  • It is also popular to establish a multi-output GP model by linearly combining the latent GPs（线性组合潜在的高斯过程来建立多输出的高斯模型也是常用方法）

B. Complexity Reduction via Sparse Approximation

  • The computational cost of performing exact inference with multi-output GP models would become prohibitive(过高的) when the training data set is large

    • Main Reason: the need of inverting a large covariance matrix

  • 两种集成了稀疏逼近的卷积多输出GP回归的近似推理技术

    • 函数形式类似于最初针对单输出GPs提出的两种近似：

      • PITC, partially independent training conditional (approximations)
      • FITC, fully independent training conditianal (approximations)

    • Relying on a set of inducing points, whose number is usually much smaller than that of the training data points

    • The use of inducing points reduces the exact GP inference to learning the posterior over the inducing points and model hyperparameters, resulting in significant reduction in the computational cost.（使用诱导点将精确GP推理简化为学习诱导点的后验和模型的超参数，使计算成本显著降低）

    • 稀疏近似被引入多输出GP回归中，使得协同多输出GP模型（COGP)和变分依赖多输出GP动态系统（VDM-GPDS）出现

      • 前者结合多个潜在稀疏GPs产生相关输出
      • 后者基于卷积多输出GPs模型

Contributions

  • Online regression and hyperparameter learning
  • Approximate model regression
  • Performance verification

C. MPF-based Model Regression and Learning

在前两小节中提出的模型预测和近似回归算法都假设超参数是不变的。然而，除了诱导点值之外，训练所考虑的稀疏多输出GP模型所需的模型超参数也需要从顺序到达的数据中学习得到。诱导点与给定超参数时的量测 $\mathbf{y}_t$ 线性相关（见 $\eqref{21a}$ 和 III-A 结尾处的讨论）。然而，超参数和 $\mathbf{y}_t$ 是线性无关的，这使得分析推导它们的后验非常困难。因此，诱导点和超参数的联合推断是一个线性与非线性混合的贝叶斯滤波问题。我们将MPF用于这个滤波任务，以实现稀疏多输出模型的在线回归与学习。

MPF算法使用算法2在 $t$ 时刻更新 $k$ 个粒子。可以看出，一旦获得新的数据集 ${ \mathbf{X}_t, \mathbf{y}_t }$ ，MPF首先更新每个粒子的超参数估计（算法2第3行）。然后，MPF使用更新后的模型超参数和算法1中给出的近似回归技术来精确每个粒子的诱导点的后验（算法2第4行）。这样，MPF实现了诱导点和模型超参数的联合推断。注意，算法1在MPF的每次迭代中运行 $K$ 次，因为它对难以分析处理的诱导点和模型超参数联合后验使用了经验近似（见 $\eqref{48}$）。

注意，算法2的第5行与使用量测似然 $\mathcal{p} (\mathbf{y}t \mid \mathcal{D}{0:t-1})$ 更新粒子权重的标准MPF不同，它使用了下界ELBO $L_t(\theta_t^k)$ 来进行粒子权重更新，这来自于更新诱导点 $\mathbf{h}$ 的后验时采用的变分近似（见 $\eqref{39}$ ）。通过粒子重采样（算法2的第8行），使用ELBO选择不同的稀疏多输出GP模型。

当在实验中实现MPF时，采用了基于有效样本大小（ESS，effrctive sample size）的残差重采样。当在线训练停止时，MPF算法输出权重最大的粒子。这与稀疏多输出模型相对应，其超参数由输出粒子的 $\theta_t^k$ 给出，其均值和协方差由输出粒子的 $\mu_t^{\mathbf{h},k}$ 和 $\mathbf{C}_t^{\mathbf{h},k}$ 给出。

在存在 $Q$ 个端节点和一个中心节点时，算法2中的MPF可以以如下分布式方式实现。每个端节点对所有 $K$ 个粒子执行第 $j$ 个latent函数的近似回归（见算法1的2-5行和10-13行）。中心节点融合来自端节点的结果，并在执行粒子重采样之后，返回更新的超参数估计 $\theta_t^k$ 以及诱导点的均值和协方差。

在上述的基于MPF的训练算法中，当训练数据点的数量 $Nt$ 较小，满足 $PN_t \lt M$ 时，最耗时的部分是在更新模型超参数和用于计算ELBO $\eqref{45}$ 的 $(\mathbf{C}{t-1}^{\mathbf{h}_j})^{-1}$ 之后 $\eqref{15}$ 中 $\mathbf{K}_j^{-1} (\mathbf{Z}_j, \mathbf{Z}_j)$ 的计算。因此，上述提出的在线训练技术的计算成本为 $\mathcal{O}(M^3)$ 。

高斯过程（Gaussian process）

在概率论和统计学中，高斯过程是观测值出现在一个连续域（例如时间或空间）的随机过程。在高斯过程中，连续输入空间中每个点都是与一个正态分布的随机变量相关联。此外，这些随机变量的每个有限集合都有一个多元正态分布，换句话说他们的任意有限线性组合是一个正态分布。高斯过程的分布是所有那些（无限多个）随机变量的联合分布，正因如此，它是连续域（例如时间或空间）上函数的分布。

高斯过程被认为是一种机器学习算法，是以惰性学习方式，利用点与点之间同质性的度量作为核函数，以从输入的训练数据预测未知点的值。其预测结果不仅包含该点的值，而同时包含不确定性的资料——它的一维高斯分布（即该点的边际分布）。

可以把高斯过程看成多元正态分布的无限维广义延伸。

定义

一统计学分布定义为 ${ Xt, t \in T }$ 是一个高斯过程，当且仅当对下标集合 $T$ 的任意有限子集 $t_1,…,t_k$ ，满足 $X{t1,…,t_k} = (X{t1},…,X{tk})$ 是一个多元正态分布，这等同于说 $(X{t1},…,X{t_k})$ 的任一线性组合是一单变量正态分布。

563.二叉树的坡度

2020-12-07

链接

563. 二叉树的坡度 - 力扣（LeetCode） (leetcode-cn.com)

题目描述

给定一个二叉树，计算 整个树 的坡度 。

一个树的 节点的坡度 定义即为，该节点左子树的节点之和和右子树节点之和的 差的绝对值 。如果没有左子树的话，左子树的节点之和为 0 ；没有右子树的话也是一样。空结点的坡度是 0 。

整个树 的坡度就是其所有节点的坡度之和。
